# Maternal Health & Vaccination Guidance System 🩺🤰

A full-stack web application developed for my academic thesis to track pregnancy progress and predict health risks.

## 🚀 Live Project
**Website Link:** [https://pregnancy-health-tracker.onrender.com](https://pregnancy-health-tracker.onrender.com)

## 🌟 Key Features
- **Pregnancy Tracking:** Calculates weeks and trimesters based on LMP.
- **Risk Prediction:** Categorizes health risks (Low/Mid/High) using Age, BP, and Sugar levels.
- **Vaccination Schedule:** Automated alerts for essential vaccines like TT.
- **Expert Q&A:** Searchable database for maternal health guidance.

## 🛠️ Technology Stack
- **Backend:** Python (Flask)
- **Database:** PostgreSQL (Render)
- **Deployment:** Render CI/CD
